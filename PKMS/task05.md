# Task 05

你学会了增量更新，但这时候又有新的麻烦了。

我希望所有的 HTML 中结构稍微做一下变化。

## 任务说明：

有的时候，我们还是需要支持全量更新，比如，在数据没变化的情况下，我们改动了 HTML 的结构。

现在我们在每个 HTML 的右侧内容下方，增加一个点赞的交互（空心大拇指），点击后，这个按钮就会变成实心的。

但是我现在不变更数据，按照上个任务实现的逻辑，现在运行脚本，因为数据没有变化，所以不会触发任何一个 HTML 更新。

怎么办呢？

我们需要在你的脚本上，增加一个执行方式，当我在运行脚本的时候，后面加一个参数，比如 -force 或者 -f，就表示强制全部更新。

来吧，试试实现这个强制更新，完成这个小功能，我们即将进入下一个阶段。

## 练习建议

- 尽可能不使用任何三方的库和框架，相关三方库、框架的练习会在后面有布置安排。
- 完全按照功能描述实现，不用额外做东西。
- 根据你实际情况，给自己限定一个完成时间。
